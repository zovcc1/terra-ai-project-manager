الخطوات التفصيلية
1
أرسل أمرًا بلغة عادية POST /api/ai/command
اكتب ما تريده كأنك تكلم شخصًا. الذكاء الاصطناعي يفهم العربية والإنجليزية.
"أنشئ مهمة جديدة اسمها 'مراجعة الكود' وعيّنها لأحمد"
"غيّر حالة مهمة قاعدة البيانات إلى مكتملة"
2
الذكاء الاصطناعي يحلل ويجهّز الإجراء
يحدد النظام ماذا يجب أن يفعل ويُنشئ إجراءً معلقًا ينتظر موافقتك قبل التنفيذ.
نوع: CREATE
العنوان: مراجعة الكود
المُعيَّن له: أحمد
الحالة: TODO
3
تأكيد أو رفض الإجراء POST /api/ai/confirm/{id}
قبل تنفيذ أي شيء، النظام يعرض عليك ما سيحدث ويطلب موافقتك.
موافق — نفّذ
رفض — إلغاء
4
إدارة الإجراءات المعلقة GET /api/ai/pending-actions
يمكنك مراجعة جميع الإجراءات التي تنتظر قرارك، وحذف أي إجراء لا تريد تنفيذه.

الإجراء النقطة الطريقة
إرسال أمر /api/ai/command POST
عرض المعلقة /api/ai/pending-actions GET
موافقة / رفض /api/ai/confirm/{actionId} POST
حذف معلقة /api/ai/pending-actions/{actionId} DELETE
ملاحظة أمان: النظام لا ينفذ أي إجراء تلقائيًا — كل شيء يمر بموافقتك أولًا. حتى لو أرسلت أمرًا خاطئًا، يمكنك الرفض قبل التنفيذ.
